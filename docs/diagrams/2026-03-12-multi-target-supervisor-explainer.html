AgentCore Identity / Multi-Target

Supervisor + Subagents + MCP Gateway

Runtime now routes logical actions to concrete tools across Atlassian and Google targets. If a tool is missing, it returns deterministic fallback metadata instead of blind calls.

Runtime: Gateway: Protocol: 2025-11-25

Execution Flow

1. User RequestComplex request mixes Jira + Calendar goals.
2. SupervisorBuilds action plan with logical tools.
3. RouterMatches tool suffix and subagent capabilities.
4. Gateway MCPRuns tools/list / tools/call.
5. Provider APIsAtlassian and Google execute with user-bound OAuth.
