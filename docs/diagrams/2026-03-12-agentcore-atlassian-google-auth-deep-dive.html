flowchart LR

  COG --> RT[AgentCore Runtime Supervisor]

  GW --> TV[Token Vault OAuth providers]
  TV --> ATL_OAUTH[Atlassian OAuth 3LO]
  TV --> GGL_OAUTH[Google OAuth 2.0]
  GW --> ATL_TARGET[atlassian-openapi-dev3]
  GW --> GGL_TARGET[google-calendar-openapi-dev3]
  ATL_TARGET --> ATL_API[Atlassian APIs]
  GGL_TARGET --> GGL_API[Google Calendar API]
  RT --> OUT[Final merged answer]

Message Lifecycle (Request to Final Answer)

Request execution across supervisor and subagents
flowchart LR
  A1[User request arrives] --> A2[Supervisor creates logical plan]
  A2 --> A3[Router checks tools list]
  A3 --> A4{Required tool exists}
  A4 -- no --> A5[Return TOOL_NOT_AVAILABLE with alternatives]
  A4 -- yes --> A6[Execute Atlassian step]
  A6 --> A7{Delegated token present}
  A7 -- no --> A8[Receive consent URL code -32042]
  A8 --> A9[User approves consent]
  A9 --> A10[CompleteResourceTokenAuth]
  A10 --> A6
  A7 -- yes --> A11[Execute Google step]
  A11 --> A12[Merge outputs and return final answer]
